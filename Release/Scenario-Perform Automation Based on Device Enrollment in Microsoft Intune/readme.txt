
This directory contains supporting code for "Scenario: Perform Automation Based on Device Enrollment in Microsoft Intune"
https://gregramsey.net/2020/03/18/scenario-perform-automation-based-on-device-enrollment-in-microsoft-intune/

DeviceEnrollment_SendEmail - LogicApps Template-Sanitized.json is a template that works through the scenario to send an email of device information. **It's not a template that you can directly deploy in Azure. You must first replace all the variables surrounded by ##. For example, ##GRAPHAUTH_CLIENT_ID## is should be the client ID for MS Graph.
